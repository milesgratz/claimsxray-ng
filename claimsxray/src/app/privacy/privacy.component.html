Privacy

This privacy statement explains the data Claims X-Ray processes, how it processes it, and for what purposes.

Personal data

Claims X-Ray token parsing occurs on the user browser and the resulting information does not leave the client machine. Token data from SAML and WS-Fed protocols are sent to the SAML Proxy Service that neither decodes the token, nor stores its content in any persistent store.

Therefore, personal data is NOT collected by the application.

Cookies and similar technologies

Claim X-Ray uses cookies and similar technologies to analyze how the application performs, and to generate statistics about token requests. It also uses “web beacons” to help deliver cookies and gather usage and performance data. The logon URL is solely used to identify the identity provider used to create the token; the raw data is kept for a maximum of 45 days.

For more information about how Claims X-Ray works, check the application architecture document.
